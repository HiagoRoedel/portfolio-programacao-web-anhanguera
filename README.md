💻 Portfolio Programação Web – Anhanguera

Este repositório contém o desenvolvimento de uma aplicação web prática realizada na disciplina de Programação Web, com o objetivo de aplicar conceitos fundamentais de integração entre front-end e back-end utilizando PHP.

📌 Sobre o Projeto

A aplicação permite que o usuário descubra seu signo zodiacal a partir da data de nascimento informada em um formulário.

O sistema realiza:

Captura de dados via formulário HTML
Processamento com PHP
Leitura de dados estruturados em XML
Exibição dinâmica do resultado
🚀 Tecnologias Utilizadas
HTML5
CSS3
Bootstrap 5
PHP
XML
XAMPP (ambiente local)
Visual Studio Code

⚙️ Como Executar o Projeto
Instale o XAMPP

Coloque o projeto dentro da pasta:

xampp/htdocs/
Inicie o Apache pelo painel do XAMPP

Acesse no navegador:

http://localhost/portfolio-programacao-web-anhanguera
🧠 Funcionamento
O usuário insere sua data de nascimento no formulário
O sistema envia os dados via método POST
O PHP processa a informação
O arquivo signos.xml é lido
A aplicação identifica o signo com base no intervalo de datas
O resultado é exibido na tela
